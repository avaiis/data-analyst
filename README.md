# Bike Sharing Dataset - Data Analysis Project

## Project Overview
Proyek ini merupakan analisis data peminjaman sepeda menggunakan Bike Sharing Dataset.
Analisis mencakup pengaruh musim & cuaca terhadap peminjaman harian, pola peminjaman per jam,
serta segmentasi waktu menggunakan teknik clustering/binning.
